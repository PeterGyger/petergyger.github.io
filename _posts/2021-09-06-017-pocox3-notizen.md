# Notizen zum ersten Kontakt mit einem chinesischen Handy  

Ein Smartphone ist für mich dasselbe wie ein Auto oder ein Computer. Zuverlässigkeit, vernünftiger Preis und Usability stehen da im Vordergrund.  

Von Black Berry, zu Windows Phone kam ich zu Android Geräten. Letztes Wochende war es an der Zeit, nach mehreren Jahren mit Samsung Geräten zu einem chinesischen Gerät zu wechseln.  

Anders als Huawei, sind die Xiaomi Geräte nicht vom Google Bann betroffen. D.h. man kann die Google Dienste nutzen.

# Ein Android ohne Google?

Bereits früher hatte ich Versuche unternommen, ein Android Gerät ohne Google zu betreiben. Nach einiger Zeit gab ich dann jeweils entnervt auf. Diesselbe Geschichte, wie der Versuch im (Berufs-) Alltag von Windows auf Linux zu wechseln. Produktivität ist nun mal meine erste Priorität.  

Sag niemals nie bzw. auch der längste Weg beginnt mit den ersten Schritten. So starte ich 2021 einen weiteren Versuch meine Unabhängigkeit gegenüber Google zu stärken.
In diesem Post halte ich die verschiedenen Etappen dieser Reise fest. 

# Kickoff

Das Samsung wandert in die Schublade und los geht es. Zuerst die Blotware entfernen. Dafür muss das Gerät nicht gerootet werden. Eine der Möglichkeiten ist „Android Debug Bridge“ (ADB). Die Android Debug Bridge stellt einen Teil des [Android SDK](https://www.droidwiki.org/wiki/Android_SDK) dar. Natürlich ist die Bloatware wieder da, wenn man das Gerät wieder resetet.  

## Xiaomi ADB/Fastboot Tools

Wer nicht mit der CLI die einzelnen ADB Befehle absetzen will, kann ein komfortables GUI verwenden. Für die Android Geräte der Marke Xiaomi hat Gergely Szaki ein einfaches Tool entwickelt: [Xiaomi ADB/Fastboot Tools (XAF Tools)](https://github.com/Szaki/XiaomiADBFastbootTools).  

Wenn man Java auf seinem Computer nicht installiert hat, findet man in der ReadMe Datei des GitHub Repositorys, im Abschnitt FAQ & Troubleshooting den Hnweis wie Java zu installieren ist. Auf dem Handy muss das USB Debugging aktiviert sein. In der ReadMe Datei ist auch dieser Vorgang Schritt für Schritt erklärt. Danach das Handy über USB mit dem Computer verbinden. 

Auf einem Computer mit einem Windows Betriebssytem kann auf der Kommandozeile die Applikation wie folgt gestartet werden:  

<p style="color: teal; font-family: 'Courier New', Courier; text-align: justify; margin-left:10px;">

C:\Program Files\Java\jdk-17\bin\java.exe -jar XiaomiADBFastbootTools.jar  

</p>

Der Pfad der exe Datei kann ggf. anders lauten. In diesem Beispiel oben, ist die Datei XiaomiADBFastbootTools.jar im aktuellen Verzeichnis. Im mittleren Fenster sieht man die vorinstallierten Apps und kann selektiv löschen:  

![Xiaomi ADB/Fastboot Tools](/images/17-adbtool.png)
